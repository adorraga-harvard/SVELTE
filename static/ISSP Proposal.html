Proposal for Metadata-Driven Automation in ISSP Evaluation

Submitted by: Amador Raga, Computer Scientist
Date: August 2025

Executive Summary

Following the release of MITHI Advisory No. 2025-02, the DICT endorsement requirement for Information Systems Strategic Plans (ISSPs) has been officially lifted. Agencies now submit ISSPs directly to the MITHI Steering Committee Secretariat, streamlining the process but introducing new challenges in volume, consistency, and strategic alignment.

“We trust agencies to innovate responsibly. MITHI must now evolve to support that trust.”
— DICT Secretary Henry Aguda, April 2025

Problem Statement

  • Manual review is unsustainable: With DICT no longer vetting ISSPs, MITHI faces a surge in submissions.
  • Inconsistent formats: ISSPs vary widely, making comparative analysis difficult.
  • Strategic misalignment risk: Projects may diverge from national ICT priorities without structured oversight.

Strategic Opportunity

This shift presents a unique opportunity to modernize the ISSP evaluation process through metadata-driven automation, ensuring scalability, transparency, and alignment with digital governance goals.

Proposed Solution

1. Metadata Schema for ISSPs

  • Project type (e.g., infrastructure, citizen services, cybersecurity)
  • Budget and funding source
  • Agency priority level
  • SDG alignment
  • Technology stack and dependencies

2. Automated Scoring Engine

  • Strategic relevance
  • Technical feasibility
  • Budget efficiency
  • Interoperability potential

3. Reviewer Dashboard

  • Real-time filtering and ranking
  • Commenting and feedback tools
  • Exportable summaries for DBM integration

Technical Stack

  • Database: EdgeDB (schema-driven, relational)
  • Frontend: SvelteKit + Tailwind CSS
  • Server: NGINX with dynamic routing
  • Containerization: Docker Compose for modular deployment

Implementation Timeline

Phase Timeline Description
Prototype Q3 2025 Develop metadata schema and scoring engine
Pilot Q4 2025 Test with select agencies (e.g., DOH, DepEd, DSWD)
Rollout Q2 2026 Full integration into MITHI review process

Request for Endorsement

This proposal seeks:

  • Technical endorsement from DICT to align with national ICT standards
  • Policy endorsement from DBM to integrate metadata scoring into budget tagging and review
